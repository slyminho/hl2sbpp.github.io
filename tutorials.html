Installation

Installing on Android

To install Half-Life 2: Sandbox++ on Android devices, follow these steps:

  • Install stable release archive and APK at HL2:SB++ GitHub page.
  • Put the folder from hl2sbpp.zip archive into your srceng folder and install the hl2sbpp APK.
  • If the folder name is "hl2sbpp-dev", rename it to "hl2sbpp".

Installing on PC

Half-Life 2: Sandbox++ includes versions for Linux, Windows and MacOS. Follow the steps below to install on PC:

  • Install stable release archive and PC binaries at HL2:SB++ GitHub page.
  • Put the folders from hl2sbpp.zip and PC Binaries.zip into your srceng folder.
  • If the folder name is "hl2sbpp-dev", rename it to "hl2sbpp".
  • Start the game using hl2sbpp_launcher.exe.

Server

Creating your server

After you will install Half-Life 2: Sandbox++, you wanna create your first server.

Press "Create server" button. You will see a little collection of the maps.

You can click the image of the maps to select the one you have clicked.

Or you can click "Game maps" button to see a list of singleplayer Half-Life 2 maps, Counter-Strike: Source maps and etc...

Like with maps section, you can click the image of the maps to select the one you have clicked.

Now, you can press "OK" button to start the server or head over to the server settings.

Now, click "Server Settings"

There you can configure server name, max count of players, password and gamemode.

If you're up to playing with your friend, you should set maxplayers count from one to 32. Friend will not be able to join if maxplayers count is 1.

You can set server hostname, maximum player count, password (if needed) and gamemode.
There are several gamemodes:

  • Default: Sandbox gamemode. You will spawn with HL2 Weapons, toolgun and physgun.
  • Sandbox: Same as default gamemode.
  • Campaign: Co-op gamemode. Play HL2 or other singleplayer maps with your friends or alone.
  • Melon Drive: You will spawn as a melon. You can't do anything but only move as a melon.
  • Deathmatch: Deathmatch gamemode from Half-Life 2: Deathmatch. Half-Life 2: Sandbox++ things are disabled here.

When you will finish configuring the server settings, you can press "OK" button in the bottom left corner to start the game.

Installing Addons

How to Install Addons

To install addons in Half-Life 2: Sandbox++, follow these steps:

  • Open your hl2sbpp folder.
  • Inside it, create a new folder named Addons (if it doesn't already exist).
  • Place your addon folders inside the Addons directory.
  • Make sure each addon follows this structure:

    Addons/AddonName/models, lua, materials, maps, etc.
  • Restart the game to load the addons.

If the addon is not working, check if the folder structure is correct and not nested incorrectly.

Multiplayer via ZeroTier

How to Play Multiplayer

This process can seem complicated at first, but once it's set up, it becomes very easy to use.

Required:

  • ZeroTier One app
  • A decent phone (if the host device is weak, the server may lag)
  • HL2SBPP (recommended over Universal Source Patch)

Host Setup

  • Open ZeroTier and create or join a network.
  • Share the Network ID with your friend (or let them scan the QR code).
  • Go to the ZeroTier website and authorize your friend's device.
  • Find your Managed IP inside the app and send it to your friend.
  • Launch the game and create a server.
  • Set the maximum number of players in server settings.
  • Start the server.

Client (Friend) Setup

  • Make sure you are connected to the same ZeroTier network.
  • Open the game console.
  • Type the following command:
    • connect (ip)
  • Replace (ip) with the host's Managed IP.

Common Issue

A common error is: "failed after 4 tries".

If this happens, check:

  • The IP address is correct
  • The friend is authorized in ZeroTier
  • Both devices are connected to the same network
  • The server is running properly
